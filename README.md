## Weather Forecast Project
## About the Project
This is a simple weather forecast web application where users can search for any city and get real-time weather details. The goal of this project is to understand how real-world applications fetch and display live data using APIs. Weather apps are useful because they provide timely information like temperature, humidity, and conditions which help users plan their daily activities better .

## Tech Stack

HTML, CSS, JavaScript

Vite (for fast development)

Weather API (for real-time data)

## Features

Search weather by city name

Displays temperature and basic weather details

Simple and clean user interface

Fast loading using Vite

## How to Run

Clone the repository

Open the project folder

Run npm install

Start the app using npm run dev

Open the local server link in your browser

Handling user input and updating UI dynamically
